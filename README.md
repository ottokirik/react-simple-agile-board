# Dashboard

Проект по реализации простой доски задач с эмуляцией серверного взаимодействия

Первое знакомство с mobx-state-tree

## Библиотеки использованные в проекте

- React
- Mobx, mobx-state-tree, mobx-react-lite
- Material UI
- React-beautiful-dnd
- Json-server
